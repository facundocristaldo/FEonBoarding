Semantic HTML

Los tags semanticos de html ayudar a los motores de busqueda, lectores de pantallas, a poder entender y comprender mejor la estructura del sitio

Algunos de estos tags semanticos son:

  • article
  • main
  • aside
  • details
  • sumary
article

Represents a selfcontained segment that can be reused withould losing context o meaning.

main

Represents the dominant content of the body.

aside

Represents a portion of the document thatÂ´s indirectly related to the main content. Frequently used for sidebars.

details

Creates a disclosure widget in with information is visible only when the widget is toggled into an open state. (used for creating this site)

sumary

Specifies the summary or caption for the details tag

footer

Represents a footer for its nearest parent

header

Represents introductory content for a section

section

Represents a generic standalone section of a document

time

Represents a specific period of time. Has a datetime attribute to translate datetime to machine-readable format
